# my NeoVim configuration

Started from [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim)

## set up

1. `git clone https://github.com/moriT958/config.nvim.git "$HOME/config.nvim"`
2. `cd ~/config.nvim`
3. `make link`
